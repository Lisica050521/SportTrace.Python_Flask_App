# SportTrace

**SportTrace** - это веб-приложение, разработанное на **Flask Python**, предназначенное для учета и отслеживания спортивных достижений пользователей. Это приложение позволяет пользователям записывать информацию о своей физической активности и отслеживать прогресс.

## Функциональность

- **Регистрация и аутентификация**: Пользователи могут создать учетную запись и войти в систему для доступа к своим данным.
- **Добавление новых достижений**: Через удобную форму ввода пользователи могут внести информацию о своих спортивных достижениях.
- **Просмотр статистики**: Пользователи могут отслеживать свои достижения в виде графиков и таблиц, отображающих изменения в их активности.
- **Установка целей и отслеживание прогресса**: Пользователи могут устанавливать собственные цели и отслеживать их достижение в приложении.

## Инструменты

- **Flask Python**: Для создания веб-приложения и обработки HTTP-запросов.
- **SQLAlchemy**: Для работы с базой данных SQLite и хранения информации о достижениях пользователей.
- **HTML/CSS**: Для создания пользовательского интерфейса и стилизации веб-страниц.
- **JavaScript**: Для создания интерактивных элементов и отображения графиков статистики.
- **Chart.js**: Библиотека JavaScript, использована для создания графиков и визуализации данных.

## Результаты

**SportTrace** предоставляет пользователям простой и удобный способ отслеживать свою физическую активность и прогресс в спортивных достижениях. Этот проект демонстрирует мои навыки в разработке веб-приложений с использованием Flask Python и инструментов веб-разработки, а также знание работы с базами данных и визуализации данных с помощью JavaScript и библиотеки Chart.js. Реализация функционала регистрации, добавления новых достижений и просмотра статистики позволяет пользователям эффективно управлять своей активностью и достигать поставленных целей. Все изменения визуализируются на графике.
